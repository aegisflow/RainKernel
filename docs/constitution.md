# Rain Kernel Constitution

1. Data is not stored. It is reconstructed.
2. All data is rule + variation.
3. The kernel is immutable.
4. Evolution occurs only by addition.
5. A fragment should contain the whole (when possible).
6. Structure is more important than content.
7. Compatibility is eternal.
8. The system is independent of language, media, and hardware.
9. The kernel must be comprehensible by a human.
10. The kernel must never grow in internal complexity.
